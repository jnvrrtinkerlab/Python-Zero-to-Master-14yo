# Lesson 12: Function Parameters and Return Values

## Learning Objectives
- Master function parameters (positional, multiple)
- Understand return statements and return values
- Learn about None and implicit returns
- Use return values to chain operations
- Debug parameter and return value issues

## Function Parameters Deep Dive

### Types of Parameters
1. **Positional Parameters**: Arguments assigned by position
2. **Multiple Parameters**: Functions can take multiple inputs
3. **Parameters vs Arguments**: Parameters are in definition, arguments are values passed

## Return Values

Functions can return:
- Single values
- Multiple values (tuples)
- None (implicitly)
- Complex objects

## Key Concepts

1. **Parameter Order**: Matters for positional arguments
2. **Return Statement**: Exits function and sends value back
3. **Early Returns**: Multiple return points in conditionals
4. **Tuple Unpacking**: Assign multiple returns to variables

## Common Use Cases

- Performing calculations and returning results
- Processing data and returning modified versions
- Validating input and returning success/failure
- Multiple related values from computation

## Challenge

Create a function that takes 3 parameters, performs some operation, and returns a meaningful result.

## Practice Tips

- Test functions with various inputs
- Print function results to verify
- Use meaningful parameter names
- Comment complex parameter logic
