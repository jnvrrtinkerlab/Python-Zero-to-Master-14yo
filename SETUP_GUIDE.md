# Complete Setup Guide for Python Learning
## Step-by-Step Installation for Beginners

---

## VS Code Installation

### Windows
1. Go to https://code.visualstudio.com/
2. Click Download for Windows
3. Run the installer
4. Click Install

### Mac & Linux
1. Go to https://code.visualstudio.com/
2. Download for your system
3. Follow installation steps

---

## Python Installation

### Windows
1. Go to https://www.python.org/downloads/
2. Click Download Python
3. **IMPORTANT**: Check "Add Python to PATH"
4. Click Install Now

### Mac & Linux
1. Python may be pre-installed
2. Open Terminal and type: python3 --version
3. If not installed, download from https://www.python.org/

---

## Verify Installation

1. Open VS Code
2. Press Ctrl + ` (backtick) to open Terminal
3. Type: python --version
4. You should see a version number

---

## Install VS Code Extensions

### Python Extension
1. Click Extensions (left sidebar)
2. Search for: Python
3. Click Install

### Pylance
1. Search for: Pylance
2. Click Install

### Code Runner
1. Search for: Code Runner
2. Click Install

### Better Comments
1. Search for: Better Comments
2. Click Install

---

## Setup GitHub Copilot (AI Assistant)

1. Create account at https://github.com
2. In VS Code Extensions, search for: GitHub Copilot
3. Click Install
4. Sign in with your GitHub account
5. Now you can use AI to help write code!

---

## Create Your First Program

1. Create a folder: My-Python-Journey
2. Open it in VS Code
3. Create file: hello_world.py
4. Type:

```python
print("Hello World!")
print("I'm learning Python!")
```

5. Click Play button to run
6. See your output in Terminal!

---

## Troubleshooting

- Python not found? Restart VS Code
- Check "Add Python to PATH" was selected during installation
- Can't run code? Install Code Runner extension
- Need help? Ask in the GitHub discussions!
